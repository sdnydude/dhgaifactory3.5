B: Collapsible Sidebar

Icons-only by default (60px), expands on hover or click. More content space. Progress bars on project cards. Grid layout.

Both layouts include:

  • Projects — board with all active projects, pipeline progress, status badges
  • Inbox — review queue filtered by role (orange badge = items needing attention)
  • Agents — live monitor for running agents + spawned subagents
  • Chat — direct agent conversation (existing functionality)
  • Dark/light toggle + Settings at bottom

Click a card to select your preference, then respond in the terminal.
