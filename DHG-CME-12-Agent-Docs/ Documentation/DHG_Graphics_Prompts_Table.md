## Design Specifications (DHG Brand System v1.0)

**DHG Brand Colors (60-30-10 Rule):**
- **Graphite** #32374A (60% - body text, structural elements)
- **Purple** #663399 (30% - headers, links, brand identity, interactive elements)
- **Orange** #F77E2D (10% - CTAs, highlights, emphasis, energy points)

**Division Colors (Use When Representing Divisions):**
- DHG AI: #6B5CE7 primary, #A78BFA secondary
- DHG CME: #059669 primary, #34D399 secondary
- DHG Studios: #DC2626 primary, #F87171 secondary
- DHG Productions: #D97706 primary, #FBBF24 secondary
- Streamcubation: #0891B2 primary, #22D3EE secondary

**Semantic Colors (Use for Functional Meaning):**
- Success: #2EAD6B (confirmations, profits, positive metrics, growth)
- Success Light: #D4EDDA (backgrounds, subtle highlights)
- Warning: #E9A430 (caution states, alerts, review gates)
- Warning Light: #FFF3CD (backgrounds)
- Critical: #B3261E (errors, costs, risks - use sparingly)
- Critical Light: #F8D7DA (backgrounds)
- Info: #4A90A4 (informational messages, secondary actions)
- Info Light: #D1ECF1 (backgrounds)

**Neutral Palette:**
- Neutral-50: #FAFAFA (lightest backgrounds)
- Neutral-100: #F4F4F5 (card backgrounds)
- Neutral-200: #E4E4E7 (borders, dividers, subtle grid lines)
- Neutral-300: #D4D4D8 (disabled backgrounds, connector lines)
- Neutral-400: #A1A1AA (placeholder text, disabled icons)
- Neutral-500: #71717A (secondary text, captions)
- Neutral-600: #52525B (body text alternate)
- Neutral-700: #3F3F46 (headings alternate)
- Neutral-800: #27272A (high contrast text)
- Neutral-900: #18181B (maximum contrast)

**Backgrounds & Surfaces:**
- Background Light: #FAF9F7 (main canvas, warm off-white)
- Background Dark: #1A1D24 (dark mode, overlays)
- Surface: #FFFFFF (cards, panels, elevated elements)

**Typography (DHG Standard):**
- **Font Family:** 'Inter', -apple-system, BlinkMacSystemFont, sans-serif
- **Monospace:** 'JetBrains Mono', 'Fira Code', monospace
- **Weights:** Light (300), Regular (400), Medium (500), Semibold (600), Bold (700), Extrabold (800)
- **Sizes:** xs (12px), sm (14px), base (16px), lg (18px), xl (20px), 2xl (24px), 3xl (30px), 4xl (36px), 5xl (48px)
- **Line Heights:** 1.6 for body text, 1.2-1.3 for headers, 1.5 for captions

**Spacing (4px Base Unit):**
- space-1: 4px | space-2: 8px | space-3: 12px | space-4: 16px
- space-5: 20px | space-6: 24px | space-8: 32px | space-10: 40px
- space-12: 48px | space-14: 56px | space-16: 64px | space-20: 80px

**Border Radius:**
- sm: 4px (data elements, small UI)
- md: 8px (cards, buttons, standard)
- lg: 12px (containers, panels)
- xl: 16px (hero elements)
- 2xl: 24px (large containers)
- full: 9999px (pills, badges)

**Shadows (DHG Official):**
- sm: 0 1px 2px rgba(50, 55, 74, 0.05)
- md: 0 4px 6px rgba(50, 55, 74, 0.1)
- lg: 0 10px 15px rgba(50, 55, 74, 0.1)
- xl: 0 20px 25px rgba(50, 55, 74, 0.15)

**Iconography Specs:**
- Style: Outlined stroke (not filled)
- Stroke Weight: 2px
- Caps/Joins: Rounded
- Sizes: 16px (inline), 20px (small UI), 24px (default), 28px (medium), 32px (emphasis), 48px (large), 64px (hero)
- ViewBox: 24×24 (default)

**DHG Brand Style Guidelines:**
- Clean, modern business aesthetic with professional polish
- Purple #663399 establishes brand presence (30% usage - headers, interactive elements, brand moments)
- Orange #F77E2D provides energy and emphasis (10% usage - CTAs, highlights, action points)
- Graphite #32374A provides structure and readability (60% usage - text, borders, structure)
- Division colors used when representing specific DHG divisions
- Semantic colors (Success, Warning, Critical, Info) used for functional meaning
- Minimal decoration, maximum clarity
- Generous white space using 4px spacing scale
- Consistent visual hierarchy
- All graphics use #FAF9F7 background (DHG light mode)
- Text always in Graphite #32374A or Neutral-500/700 for hierarchy
- Icons use 2px outlined stroke with rounded caps
- Border radius maintains DHG standard (8-12px for most elements)
- Shadows add subtle depth without distraction
