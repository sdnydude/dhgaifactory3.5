# DHG Style Guide

## Usage
This style guide applies to ALL Digital Harmony Group work. Import the CSS file or use these variables directly.

## Light Mode Colors

| Variable | Hex | Usage |
|----------|-----|-------|
| `--dhg-background` | #FAF9F7 | Page background (warm off-white) |
| `--dhg-surface` | #FFFFFF | Cards, panels |
| `--dhg-surface-elevated` | #FFFFFF | Elevated elements |
| `--dhg-text-primary` | #32374A | Main text (dark blue-gray) |
| `--dhg-text-secondary` | #71717A | Secondary text |
| `--dhg-text-placeholder` | #A1A1AA | Placeholder text |
| `--dhg-border` | #E4E4E7 | Borders |
| `--dhg-border-focus` | #663399 | Focus state (purple) |

## Dark Mode Colors

| Variable | Hex | Usage |
|----------|-----|-------|
| `--dhg-background` | #1A1D24 | Page background (dark blue-black) |
| `--dhg-surface` | #27272A | Cards, panels |
| `--dhg-surface-elevated` | #32374A | Elevated elements |
| `--dhg-text-primary` | #FAF9F7 | Main text (warm white) |
| `--dhg-text-secondary` | #A1A1AA | Secondary text |
| `--dhg-text-placeholder` | #71717A | Placeholder text |
| `--dhg-border` | #3F3F46 | Borders |
| `--dhg-border-focus` | #A78BFA | Focus state (light purple) |

## Key Principles

1. **Warm tones**: Background is warm off-white (#FAF9F7), not pure white
2. **Purple accents**: Focus/accent color is purple (#663399 light / #A78BFA dark)
3. **Dark mode support**: All UIs must support both light and dark modes
4. **Elevated surfaces**: Use `--dhg-surface-elevated` for modals, dropdowns
