# Knowledge Base

This folder contains knowledge documents, reference materials, and curated information for the DHG AI Factory project.

## Structure

```
knowledge/
├── README.md           # This file
├── agents/             # Agent-specific knowledge
├── cme/                # CME/medical education content
├── docs/               # Documentation and guides
└── templates/          # Document templates
```

## Usage

- Add curated documents, PDFs, and reference materials
- Used by RAGFlow for retrieval-augmented generation
- Synced with Dify knowledge bases as needed

## Formats Supported

- Markdown (.md)
- PDF (.pdf)
- Text (.txt)
- JSON (.json)
- Word documents (.docx)
